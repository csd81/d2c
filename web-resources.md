# Web Resources

External resources used for comparing `d2c` against Claude Code / the
*Dive into Claude Code* paper.

## Primary Paper And Companion Materials

- https://arxiv.org/abs/2604.14228
  - arXiv abstract for *Dive into Claude Code: The Design Space of Today's and Future AI Agent Systems*.

- https://arxiv.org/html/2604.14228v1
  - HTML version of the paper.

- https://github.com/VILA-Lab/Dive-into-Claude-Code
  - Official companion repository with diagrams, reading guide, design-space notes, and community resources.

- https://zhiqiangshen.com/projects/Claude_Code_Report/Claude_Code_Report.pdf
  - PDF mirror of the paper/report.

## Detailed Architecture Analyses

- https://zainhas.github.io/blog/2026/inside-claude-code-architecture/
  - Detailed visual/narrative architecture deep dive covering startup, query loop, tool execution,
    memory, compaction, permissions, and module structure.

- https://www.penligent.ai/hackinglabs/tr/inside-claude-code-the-architecture-behind-tools-memory-hooks-and-mcp/
  - Practical security/architecture analysis focused on tools, memory, hooks, MCP, permissions,
    sandboxing, subagents, and governance.

- https://www.reddit.com/r/ClaudeAI/comments/1sa6ih3/claude_code_source_deep_dive_part_1_architecture/
  - Community reverse-engineering/source-teardown style writeup. Useful for extra detail, but treat
    with caution because it is community/leak-based material.

- https://medium.com/@roanmonteiro/the-claude-code-agentic-loop-a-complete-practitioners-guide-0b338647a3aa
  - Practitioner-oriented guide to the Claude Code agentic loop, permissions, hooks, skills,
    subagents, and MCP workflows.

- https://fluidattacks.com/blog/claude-code-ai-agents-engineering
  - Security/engineering interpretation of the paper and Claude Code architecture.
